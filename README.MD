# 职能沟通翻译助手

一个基于大模型的翻译工具，用于弥合产品经理和开发工程师之间的沟通鸿沟。

## 🎯 项目目标

- **产品经理 → 开发工程师**：将需求描述翻译成技术实现方案
- **开发工程师 → 产品经理**：将技术方案翻译成业务价值描述

## 📁 项目结构

phase-console/ # 第一阶段：控制台版本
phase-web-api/ # 第二阶段：Express + TS + HTML
phase-nuxt-fullstack/ # 第三阶段：Nuxt 全栈
phase-npm-package/ # 第四阶段：NPM 包

## 🚀 快速开始

### 第一阶段：控制台版本
```shell
bash
cd phase-console
npm install
cp .env.example .env # 配置你的 OpenAI API
npm start
```

## 🔧 技术栈

- **阶段1**: Node.js + OpenAI SDK
- **阶段2**: TypeScript + Express + HTML
- **阶段3**: Nuxt 3 (Vue 3) 全栈
- **阶段4**: 可独立使用的 NPM 包
